# 📊 Dashboard de Vendas - Excel

## 🔎 Sobre o projeto
Dashboard desenvolvido em Excel com foco em análise de vendas de assinaturas (Game Pass), incluindo indicadores de faturamento, ticket médio e perfil de clientes.

## 📈 Indicadores
- Faturamento Bruto
- Faturamento Líquido
- Ticket Médio
- Vendas por pacote
- Perfil de clientes

## 🛠️ Ferramentas utilizadas
- Excel
- Tabela Dinâmica
- Segmentação de Dados
- Gráficos dinâmicos

## 🎯 Objetivo
Demonstrar habilidades em análise de dados, organização de informações e construção de dashboards.

## 📌 Observação
Os dados são fictícios para fins de estudo.
